## Sources principales {#sources-principales}

:::: {.section}

Les sources principales utilisées pour rédiger les pages du site sont :

- [Wikipédia - Marie Curie](https://fr.wikipedia.org/wiki/Marie_Curie)
- [Nobel Prize - Marie Curie](https://www.nobelprize.org/prizes/physics/1903/marie-curie/biographical/)

## Sources pour les images {#sources-images}

:::: {.section}

Les images utilisées dans le site représentent Marie Curie, Pierre Curie, son laboratoire et l’utilisation de la radiologie.  
Elles servent à illustrer les différentes pages du mini-site.

Les images utilisées sont placées dans le dossier `IMAGES`.

::::

## Utilisation des sources {#utilisation}

:::: {.section}

Les sources ont été utilisées pour rédiger les différentes parties du site :

- la biographie de Marie Curie ;
- ses travaux scientifiques ;
- ses découvertes principales ;
- son impact dans la médecine et la société ;
- la chronologie de sa vie.

Les textes ont été reformulés afin de produire un contenu clair et adapté au projet.
